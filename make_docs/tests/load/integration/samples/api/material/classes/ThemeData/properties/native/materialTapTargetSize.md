# materialTapTargetSize property

[MaterialTapTargetSize](flutter-docs://api/material/MaterialTapTargetSize) materialTapTargetSize


Configures the hit test size of certain Material widgets.

Defaults to a [platform](flutter-docs://api/material/ThemeData/platform)-appropriate size: [MaterialTapTargetSize.padded](flutter-docs://api/material/MaterialTapTargetSize) on mobile platforms, [MaterialTapTargetSize.shrinkWrap](flutter-docs://api/material/MaterialTapTargetSize) on desktop
platforms.

## Implementation

```dart
final MaterialTapTargetSize materialTapTargetSize;
```
