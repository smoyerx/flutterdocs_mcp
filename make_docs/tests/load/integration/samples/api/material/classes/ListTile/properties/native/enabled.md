# enabled property

[bool](flutter-docs://api/dart-core/bool) enabled


Whether this list tile is interactive.

If false, this list tile is styled with the disabled color from the
current [Theme](flutter-docs://api/material/Theme) and the [onTap](flutter-docs://api/material/ListTile/onTap) and [onLongPress](flutter-docs://api/material/ListTile/onLongPress) callbacks are
inoperative.

## Implementation

```dart
final bool enabled;
```
