# minVerticalPadding property

[double](flutter-docs://api/dart-core/double)? minVerticalPadding


The minimum padding on the top and bottom of the title and subtitle widgets.

If null, then the value of [ListTileTheme.minVerticalPadding](flutter-docs://api/material/ListTileTheme/minVerticalPadding) is used. If
that is also null, then a default value of 4 is used.

## Implementation

```dart
final double? minVerticalPadding;
```
