# highlightColor property

[Color](flutter-docs://api/dart-ui/Color)? highlightColor

The highlight color of the ink response when pressed. If this property is
null then the highlight color of the theme, [ThemeData.highlightColor](flutter-docs://api/material/ThemeData/highlightColor),
will be used.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [highlightColor](flutter-docs://api/material/InkResponse/highlightColor).
