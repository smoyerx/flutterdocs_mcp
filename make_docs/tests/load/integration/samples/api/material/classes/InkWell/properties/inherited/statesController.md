# statesController property

[MaterialStatesController](flutter-docs://api/material/MaterialStatesController)? statesController

Represents the interactive "state" of this widget in terms of
a set of [WidgetState](flutter-docs://api/widgets/WidgetState) s, like [WidgetState.pressed](flutter-docs://api/widgets/WidgetState) and
[WidgetState.focused](flutter-docs://api/widgets/WidgetState).

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [statesController](flutter-docs://api/material/InkResponse/statesController).
