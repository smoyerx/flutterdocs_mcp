# onSecondaryTapCancel property

[GestureTapCallback](flutter-docs://api/gestures/GestureTapCallback)? onSecondaryTapCancel

Called when the user cancels a secondary button tap that was started on
this part of the material.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [onSecondaryTapCancel](flutter-docs://api/material/InkResponse/onSecondaryTapCancel).
