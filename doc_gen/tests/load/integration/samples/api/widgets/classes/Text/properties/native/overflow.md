# overflow property

[TextOverflow](flutter-docs://api/painting/TextOverflow)? overflow


How visual overflow should be handled.

If this is null [TextStyle.overflow](flutter-docs://api/painting/TextStyle/overflow) will be used, otherwise the value
from the nearest [DefaultTextStyle](flutter-docs://api/widgets/DefaultTextStyle) ancestor will be used.

## Implementation

```dart
final TextOverflow? overflow;
```
