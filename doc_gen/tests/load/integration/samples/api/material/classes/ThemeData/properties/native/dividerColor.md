# dividerColor property

[Color](mcp://flutter/api/dart-ui/Color) dividerColor


The color of [Divider](mcp://flutter/api/material/Divider) s and [PopupMenuDivider](mcp://flutter/api/material/PopupMenuDivider) s, also used
between [ListTile](mcp://flutter/api/material/ListTile) s, between rows in [DataTable](mcp://flutter/api/material/DataTable) s, and so forth.

To create an appropriate [BorderSide](mcp://flutter/api/painting/BorderSide) that uses this color, consider [Divider.createBorderSide](mcp://flutter/api/material/Divider/createBorderSide).

## Implementation

```dart
final Color dividerColor;
```
