# onHighlightChanged property

[ValueChanged](flutter-docs://api/foundation/ValueChanged)<[bool](flutter-docs://api/dart-core/bool)>? onHighlightChanged

being highlighted.

See further details at [onHighlightChanged](flutter-docs://api/material/InkResponse/onHighlightChanged).
