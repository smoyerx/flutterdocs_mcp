# onTapCancel property

[GestureTapCallback](mcp://flutter/api/gestures/GestureTapCallback)? onTapCancel

Called when the user cancels a tap that was started on this part of the
material.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [onTapCancel](mcp://flutter/api/material/InkResponse/onTapCancel).
