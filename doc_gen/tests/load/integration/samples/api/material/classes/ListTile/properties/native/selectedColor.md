# selectedColor property

[Color](flutter-docs://api/dart-ui/Color)? selectedColor


Defines the color used for icons and text when the list tile is selected.

If this property is null then [ListTileThemeData.selectedColor](flutter-docs://api/material/ListTileThemeData/selectedColor) is used. If that is also null then [ColorScheme.primary](flutter-docs://api/material/ColorScheme/primary) is used.

See also:

- [ListTileTheme.of](flutter-docs://api/material/ListTileTheme/of), which returns the nearest [ListTileTheme](flutter-docs://api/material/ListTileTheme)'s
[ListTileThemeData](flutter-docs://api/material/ListTileThemeData).


## Implementation

```dart
final Color? selectedColor;
```
