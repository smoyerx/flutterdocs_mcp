Create a [ThemeData](flutter-docs://api/material/ThemeData) given a set of exact values. Most values must be
specified. They all must also be non-null except for `cupertinoOverrideTheme`, and deprecated members.

This will rarely be used directly. It is used by [lerp](flutter-docs://api/material/ThemeData/lerp) to
create intermediate themes based on two themes created with the [ThemeData](flutter-docs://api/material/ThemeData) constructor.
