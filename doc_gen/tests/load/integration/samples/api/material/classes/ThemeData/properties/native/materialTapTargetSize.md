# materialTapTargetSize property

[MaterialTapTargetSize](mcp://flutter/api/material/MaterialTapTargetSize) materialTapTargetSize


Configures the hit test size of certain Material widgets.

Defaults to a [platform](mcp://flutter/api/material/ThemeData/platform)-appropriate size: [MaterialTapTargetSize.padded](mcp://flutter/api/material/MaterialTapTargetSize) on mobile platforms, [MaterialTapTargetSize.shrinkWrap](mcp://flutter/api/material/MaterialTapTargetSize) on desktop
platforms.

## Implementation

```dart
final MaterialTapTargetSize materialTapTargetSize;
```
