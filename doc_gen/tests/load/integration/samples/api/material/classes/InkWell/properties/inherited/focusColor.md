# focusColor property

[Color](mcp://flutter/api/dart-ui/Color)? focusColor

The color of the ink response when the parent widget is focused. If this
property is null then the focus color of the theme,
[ThemeData.focusColor](mcp://flutter/api/material/ThemeData/focusColor), will be used.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [focusColor](mcp://flutter/api/material/InkResponse/focusColor).
