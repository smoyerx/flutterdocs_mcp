# focusColor property

[Color](flutter-docs://api/dart-ui/Color)? focusColor

The color of the ink response when the parent widget is focused. If this
property is null then the focus color of the theme,
[ThemeData.focusColor](flutter-docs://api/material/ThemeData/focusColor), will be used.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [focusColor](flutter-docs://api/material/InkResponse/focusColor).
