# highlightShape property

[BoxShape](mcp://flutter/api/painting/BoxShape) highlightShape

The shape (e.g., circle, rectangle) to use for the highlight drawn around
this part of the material when pressed, hovered over, or focused.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [highlightShape](mcp://flutter/api/material/InkResponse/highlightShape).
