# highlightShape property

[BoxShape](flutter-docs://api/painting/BoxShape) highlightShape

The shape (e.g., circle, rectangle) to use for the highlight drawn around
this part of the material when pressed, hovered over, or focused.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [highlightShape](flutter-docs://api/material/InkResponse/highlightShape).
