# autofocus property

[bool](mcp://flutter/api/dart-core/bool) autofocus

True if this widget will be selected as the initial focus when no other
node in its scope is currently focused.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [autofocus](mcp://flutter/api/material/InkResponse/autofocus).
