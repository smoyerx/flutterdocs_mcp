# horizontalTitleGap property

[double](mcp://flutter/api/dart-core/double)? horizontalTitleGap


The horizontal gap between the titles and the leading/trailing widgets.

If null, then the value of [ListTileTheme.horizontalTitleGap](mcp://flutter/api/material/ListTileTheme/horizontalTitleGap) is used. If
that is also null, then a default value of 16 is used.

## Implementation

```dart
final double? horizontalTitleGap;
```
