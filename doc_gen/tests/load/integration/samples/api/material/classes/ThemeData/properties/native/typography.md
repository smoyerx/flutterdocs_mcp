# typography property

[Typography](flutter-docs://api/material/Typography) typography


The color and geometry [TextTheme](flutter-docs://api/material/TextTheme) values used to configure [textTheme](flutter-docs://api/material/ThemeData/textTheme).

Defaults to a [platform](flutter-docs://api/material/ThemeData/platform)-appropriate typography.

## Implementation

```dart
final Typography typography;
```
