# typography property

[Typography](mcp://flutter/api/material/Typography) typography


The color and geometry [TextTheme](mcp://flutter/api/material/TextTheme) values used to configure [textTheme](mcp://flutter/api/material/ThemeData/textTheme).

Defaults to a [platform](mcp://flutter/api/material/ThemeData/platform)-appropriate typography.
