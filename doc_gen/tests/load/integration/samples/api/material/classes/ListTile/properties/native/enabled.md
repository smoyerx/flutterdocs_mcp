# enabled property

[bool](mcp://flutter/api/dart-core/bool) enabled


Whether this list tile is interactive.

If false, this list tile is styled with the disabled color from the
current [Theme](mcp://flutter/api/material/Theme) and the [onTap](mcp://flutter/api/material/ListTile/onTap) and [onLongPress](mcp://flutter/api/material/ListTile/onLongPress) callbacks are
inoperative.

## Implementation

```dart
final bool enabled;
```
