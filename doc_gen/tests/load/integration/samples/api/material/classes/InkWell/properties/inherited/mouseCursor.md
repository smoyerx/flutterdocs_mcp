# mouseCursor property

[MouseCursor](mcp://flutter/api/services/MouseCursor)? mouseCursor

The cursor for a mouse pointer when it enters or is hovering over the
widget.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [mouseCursor](mcp://flutter/api/material/InkResponse/mouseCursor).
