# splashColor property

[Color](flutter-docs://api/dart-ui/Color)? splashColor

The splash color of the ink response. If this property is null then the
splash color of the theme, [ThemeData.splashColor](flutter-docs://api/material/ThemeData/splashColor), will be used.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [splashColor](flutter-docs://api/material/InkResponse/splashColor).
