# splashColor property

[Color](mcp://flutter/api/dart-ui/Color)? splashColor

The splash color of the ink response. If this property is null then the
splash color of the theme, [ThemeData.splashColor](mcp://flutter/api/material/ThemeData/splashColor), will be used.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [splashColor](mcp://flutter/api/material/InkResponse/splashColor).
