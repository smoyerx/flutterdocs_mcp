# hoverColor property

[Color](mcp://flutter/api/dart-ui/Color)? hoverColor

The color of the ink response when a pointer is hovering over it. If this
property is null then the hover color of the theme,
[ThemeData.hoverColor](mcp://flutter/api/material/ThemeData/hoverColor), will be used.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [hoverColor](mcp://flutter/api/material/InkResponse/hoverColor).
