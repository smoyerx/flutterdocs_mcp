# hoverColor property

[Color](flutter-docs://api/dart-ui/Color)? hoverColor

The color of the ink response when a pointer is hovering over it. If this
property is null then the hover color of the theme,
[ThemeData.hoverColor](flutter-docs://api/material/ThemeData/hoverColor), will be used.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [hoverColor](flutter-docs://api/material/InkResponse/hoverColor).
