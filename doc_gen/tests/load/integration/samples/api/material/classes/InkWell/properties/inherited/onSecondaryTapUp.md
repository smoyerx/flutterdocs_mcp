# onSecondaryTapUp property

[GestureTapUpCallback](flutter-docs://api/gestures/GestureTapUpCallback)? onSecondaryTapUp

Called when the user releases a secondary button tap that was started on
this part of the material. [onSecondaryTap](flutter-docs://api/material/InkResponse/onSecondaryTap) is called immediately after.

This property is inherited from [InkResponse](flutter-docs://api/material/InkResponse).
See further details at [onSecondaryTapUp](flutter-docs://api/material/InkResponse/onSecondaryTapUp).
