# onSecondaryTapUp property

[GestureTapUpCallback](mcp://flutter/api/gestures/GestureTapUpCallback)? onSecondaryTapUp

Called when the user releases a secondary button tap that was started on
this part of the material. [onSecondaryTap](mcp://flutter/api/material/InkResponse/onSecondaryTap) is called immediately after.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [onSecondaryTapUp](mcp://flutter/api/material/InkResponse/onSecondaryTapUp).
