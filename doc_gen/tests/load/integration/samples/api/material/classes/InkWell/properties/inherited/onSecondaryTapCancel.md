# onSecondaryTapCancel property

[GestureTapCallback](mcp://flutter/api/gestures/GestureTapCallback)? onSecondaryTapCancel

Called when the user cancels a secondary button tap that was started on
this part of the material.

This property is inherited from [InkResponse](mcp://flutter/api/material/InkResponse).
See further details at [onSecondaryTapCancel](mcp://flutter/api/material/InkResponse/onSecondaryTapCancel).
